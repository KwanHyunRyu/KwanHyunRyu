# 류관현 | Backend Developer Portfolio

> **Live Demo:** [https://kwanhyunryu.github.io](https://kwanhyunryu.github.io)

비즈니스 요구사항을 명확히 이해하고, 시스템의 안정성과 확장성을 고려하여 최적의 기술적 해결책을 설계하는 백엔드 개발자입니다. 

VDeX(VDI 원격 접속 솔루션) 및 ORIS(해외송금 중계 시스템) 등 실무 엔터프라이즈 프로젝트에서 아키텍처 설계부터 구축, 운영 환경 최적화까지 전체 라이프사이클을 주도한 경험이 있습니다.

## 🛠️ Portfolio Tech Stack
이 포트폴리오 웹사이트는 다음 기술로 구축 및 배포되었습니다.
- **Frontend:** React.js
- **Deployment:** GitHub Pages

*(저의 주력 백엔드 기술 스택인 `Java`, `Spring Boot`, `PostgreSQL` 등에 대한 상세 역량과 코드는 포트폴리오 웹사이트 본문에서 확인하실 수 있습니다.)*

## Key Experience Highlights
- **ORIS (해외송금 중계 시스템):** 한국핀테크산업협회 주관 대외 시스템의 서비스 아키텍처 설계 및 백엔드 전 구간 구축 주도.
- **VDeX (VDI 솔루션):** 신규 솔루션 기술 스택 마이그레이션 및 구축 리딩, 타 부서 및 외부 개발 그룹과의 협업 주도.
- **외환전산망 시스템:** 대규모 레거시 시스템의 안정적인 유지보수 및 시스템 정상화 담당.

## Local Installation
해당 React 프로젝트를 로컬 환경에서 확인하고 실행하기 위한 방법입니다.

```bash
# 1. 저장소 클론
$ git clone [https://github.com/KwanHyunRyu/KwanHyunRyu.github.io.git](https://github.com/KwanHyunRyu/KwanHyunRyu.github.io.git)

# 2. 디렉토리 이동
$ cd KwanHyunRyu.github.io

# 3. 의존성 패키지 설치
$ npm install

# 4. 로컬 서버 실행 (http://localhost:3000)
$ npm start
